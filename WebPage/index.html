Right Here Right Now
Prateek Garg, Peter Bekos, Hardik Prajapati, Junior Luna

Use

Functionality: Current temperature / weather, step by step directions, live road data, info graphics

How to Use:

Sidenav

  • Layer Controls
    • UIC to Museum Campus: shows data between UIC and Museum Campus
    • Overall Chicago: shows all info for chicago
    • Refresh Map: refreshes the map
    • Map View: sets map to standard map view
    • Satellite View: sets map to satellite / arial view
    • Street View: sets map to road view
    • View Items: Checking these off show the following live occurances
      • Potholes
      • Abondoned Vehicles
      • Street Lights
      • Divvy Stands
      • Crime Data
      • CTA Buses
  • Profile: about us (unfinished)
  • Messages: notifications (unfinished)
  • Settings: custom settings (unfinished)

Clicking on the map:

Clicking once on the map will put down a marker.

Clicking again will find a path for you to take

Data

  • d3: javascript library used to display data - http://d3js.org/
  • Leaflet: javascript library used for interactive map - http://leafletjs.com/
  • CTA api: gateway to real time CTA bus location and info - http://www.transitchicago.com/assets/1/developer_center/BusTime_Developer_API_Guide.pdf
  • php proxy: php script used to fetch non local data - http://benalman.com/projects/php-simple-proxy
  • yahoo proxy: used to fetch non local data - https://developer.yahoo.com/javascript/howto-proxy.html
  • jquery: javascript library used for HTML document traversal and manipulation - http://jquery.com/
  • routing machine: routing control for leaflet - http://www.liedman.net/leaflet-routing-machine/
  • divvy data
  • Weather App

Source

Github link: https://github.com/prateekgarg/RightHereRightNow
